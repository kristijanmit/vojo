## ⚙️Installation

### 🔧Install prerequisites (once for a machine)

- **Node Installation:** [Install node js](https://nodejs.org/en/download/) [Recommended LTS version]

### 🖥️Local setup

- Install dependencies

```
npm install
```

- Run locally

```
npm run dev
```

## 🔨Production Build

```
npm run build
```
